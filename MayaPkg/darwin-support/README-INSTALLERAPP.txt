Installation
============

There are 2 files in the Radeon ProRender Maya plug-in installer image:
- The RadeonProRenderMayaInstaller
- README.txt

The RadeonProRenderMayaInstaller application is used to install and uninstall the Radeon ProRender Maya plug-in.

Installation is done in multiple steps:
- A link to a web page providing a key is given (key is sent by email)
- The key is entered in the following prompt
- The installer package is then opened, files are copied and Maya modules are setup

Uninstallation is run by launching the RadeonProRenderMayaInstaller application:
- Installed files will be removed
- Afterwards, the /Users/Shared/RadeonProRender/Maya/ will be moved to the Trash and a computer password prompt will be shown

Directories
===========

Running the installation application will place files into the following directory:
- /Users/Shared/RadeonProRender/Maya/

If run successfully, the installer application will place module files into: 
- /Users/Shared/Autodesk/modules/maya/...
